# 🪖 Termux‑BASHRC
- LégionOS CYBER OPS v6
   - (ULTIMATE Military Edition)

Termux‑BASHRC est un environnement Bash militaire avancé, conçu pour Termux ARM64, optimisé pour les opérations CYBER OPS, la défense, le forensic, l’OSINT, le SIGINT, et l’audit réseau.

Ce projet fournit un .bashrc complet, moderne, tactique, inspiré des environnements cyber militaires (SOC, CERT, RensOps), avec un design LégionOS.

---

⚔️ Fonctionnalités principales

🧠 HUD tactique (v6 Ultra‑Optimisé)
Affiche en un instant :
- Charge CPU  
- RAM disponible  
- Stockage  
- IP publique  
- Ports ouverts  
- Processus suspects  

Commande :  
`bash
hud
`

---

🛡️ SOC++++ (Défense avancée)
Modules de détection et d’analyse :
- Threat Hunt (événements suspects)
- Log Hunt (analyse des logs)
- Process Scan (processus suspects)
- Net Hunt (analyse réseau)

Commandes :  
`bash
threathunt
loghunt
pscan
nethunt
`

---

🌐 OSINT intégré
Analyse rapide :
- IP (ipinfo.io)
- Domaine (dig + whois)

Commandes :  
`bash
osint_ip <IP>
osint_domain <domaine>
`

---

📡 SIGINT (Surveillance réseau)
Surveillance réseau légère :
- Ports actifs
- Logs réseau
- Connexions TCP/UDP

Commande :  
`bash
sigint
`

---

🧩 Audit réseau
Scans rapides ou complets :
`bash
scan
fastscan
stealth
fullscan
`

---

🛠️ Outils système
- Diagnostic complet  
- Décompression universelle  
- Navigation cyber  

Commandes :  
`bash
diagnostic
extract <fichier>
ops / tools / scripts / wordlists
`
